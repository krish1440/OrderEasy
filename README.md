## ✨ Features that Define Us

### 📊 Visual Business Intelligence
- **Advanced Recharts Integration**: Interactive, responsive charts showing revenue trends, order status distribution, and fulfillment gaps.
- **Real-time BI Analytics**: Dynamic mapping of backend data into visual KPI cards and data-rich tables.
- **Interactive Calendar**: Full-view delivery scheduling for better warehouse management.

### 🛡️ User Experience & Interaction
- **Resizable Sidebar**: Draggable layout logic allowing users to customize their workspace.
- **Framer Motion Animations**: Smooth page transitions, entry effects, and micro-interactions for a premium feel.
- **Skeleton Loading State**: Pulsing gray placeholders that reduce "perceived wait time" during data fetching.

### 📦 Operational Professionalism
- **Dynamic Order Forms**: Custom field injection for diverse business needs.
- **Multi-format Exports**: Direct downloads of business reports in professional Excel and PDF formats.

---

## 🛠️ Technology Stack
- **Framework**: React 18+ (Vite)
- **Language**: TypeScript
- **Styling**: Tailwind CSS, Lucide Icons
- **Visualization**: Recharts, Framer Motion
- **Networking**: Axios (with custom Service Layer)

## 🚀 Getting Started

1. **Clone & Install**:
   ```bash
   git clone [repository-url]
   cd OrderEazy/Fronted
   npm install
   ```

2. **Configuration**:
   Ensure `vite.config.ts` has the correct proxy settings for the backend port.

3. **Run Development Server**:
   ```bash
   npm run dev
   ```
